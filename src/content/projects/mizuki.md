## 项目概述

Mizuki 是一个现代化的博客主题，采用 Material Design 3 设计语言。基于 Astro 框架构建，提供了优秀的性能和用户体验。

### 主要特性

- 🎨 Material Design 3 设计语言
- 🌍 多语言国际化支持 (i18n)
- 🌙 深色模式支持
- 📱 完全响应式设计
- ⚡ 极快的页面加载速度
- 🔧 高度可定制

### 技术栈

- **框架**: Astro
- **编程语言**: TypeScript
- **样式**: Tailwind CSS
- **交互**: Svelte

该项目已完成，在 GitHub 开源，并有在线演示版本。
