## 项目概述

FolkSplash 是一个基于网络的 splash.img 可视化工具，专为 OPPO、Realme 和 OnePlus 设备设计。支持解包、预览、替换和重新打包等操作。

### 主要特性

- 🎨 Web 在线工具，无需安装
- 📦 Splash.img 解包和打包
- 👀 高清预览功能
- 🔄 图片替换功能
- 📱 适配多种设备
- ⚡ 快速处理速度

### 技术栈

- **前端框架**: React
- **编程语言**: TypeScript
- **构建工具**: Vite
- **UI 库**: Material-UI
- **状态管理**: Zustand

项目已完成，提供了一个方便的在线 splash 管理工具。
