## 项目概述

FolkPatch 是一个基于 KernelPatch 的内核级 ROOT 解决方案，提供了优雅的用户界面、APM 模块系统和 KPM 内核模块支持。

### 主要特性

- 🔧 基于 KernelPatch 框架
- 🎨 精心设计的用户界面
- 📦 APM 模块化系统
- 🔌 KPM 内核模块支持
- ⚡ 高效的内核操作

### 技术栈

- **安卓开发**: Kotlin, Java
- **底层**: Rust, C++
- **系统集成**: 内核模块

该项目持续开发中，致力于为 Android 用户提供强大的系统定制工具。
