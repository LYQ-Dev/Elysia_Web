  ## 📌 408考点重点与难点总结
  
  ### 一、硬布线控制器（组合逻辑控制器）
1. **核心思想**
   纯硬件电路（门电路+触发器）直接产生控制信号，**无存储环节**。
   控制信号 = f(操作码，时序信号，状态条件)
2. **408高频考点**
- 组成：IR、译码器、时序系统、条件判断、控制信号产生电路
- 优点：速度极快、执行效率高
- 缺点：设计复杂、修改困难、扩展性差
- 适用：RISC、高速CPU、指令集简单的处理器
  
  ### 二、微程序控制器
1. **核心思想**
   把控制信号编成**微指令**，存在**控制存储器CM**中，执行时逐条读出产生控制信号。
2. **408高频考点**
- 核心部件：CM、μIR、μAR、微地址形成逻辑
- 微指令格式：操作控制字段 + 顺序控制字段
- 编码方式：直接编码、字段直接编码、字段间接编码（必考）
- 微程序流程：取微指令 → 执行微操作 → 形成下地址
- 优点：设计规整、易修改、扩展性强
- 缺点：速度较慢（需访存）
- 适用：CISC、复杂指令集CPU
  
  ### 三、408必背对比表（选择/简答题高频）
  
  | 对比项  | 硬布线控制器      | 微程序控制器     |
  | ---- | ----------- | ---------- |
  | 实现方式 | 组合逻辑电路      | 存储逻辑（微程序）  |
  | 执行速度 | 快           | 较慢         |
  | 设计难度 | 高，指令越复杂越难   | 低，规整易设计    |
  | 扩展性  | 差，修改需改电路    | 强，只需改微程序   |
  | 应用场景 | RISC、高性能CPU | CISC、通用CPU |
